# AI Fitness Application

This project is part of my final-year dissertation.

## Description
A full-stack fitness tracking application that provides AI-assisted workout and fitness recommendations.

## Technologies
- Backend: Spring Boot (Java)
- Frontend: Angular
- Containerisation: Docker, Docker Compose

## Project Structure
- fitnessTrackerServer/ – backend service
- fitnessWeb/ – frontend application
- docker-compose.yml – multi-container configuration

## How to Run Locally
```bash
docker compose up --build
